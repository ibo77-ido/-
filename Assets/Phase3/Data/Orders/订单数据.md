# 订单数据 (Order Data)

> 汇总自 `Assets/Phase3/Data/Orders/` 下所有 OrderData资产。
> 仅抽取名称、器型、声望、银两四个字段，便于快速查阅。

## 字段说明

| 字段 | 说明 |
|------|------|
| 订单编号 | OrderData.orderID |
| 名称     | OrderData.orderName |
| 器型     | OrderData.requiredShapeID 对应的器型中文名（来自 ShapeConfigSO.nameCN） |
| 声望     | OrderData.baseReputation |
| 银两     | OrderData.baseGold |

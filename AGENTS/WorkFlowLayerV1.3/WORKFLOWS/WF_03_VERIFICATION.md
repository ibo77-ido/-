# WF_03_VERIFICATION

## Purpose
Verify task implementation against Acceptance Criteria.

## Input
CORE.md, Current Task, Evidence

## Output
Verification Report

## Status
PASS / FAIL / BLOCKED

## Procedure
Evaluate each Acceptance Criterion with supporting evidence

## Output Template
Verification Report:
- Task ID
- Task Status
- Acceptance Results [PASS/FAIL/BLOCKED] + Evidence
- Modified Files
- Risks
- Next Action

# WF_05_CLOSURE

## Purpose
Close completed work and archive project state.

## Input
TASK_QUEUE.md, STATE.json

## Output
Closure Report

## Closure Levels
Feature / Milestone / Goal / Project

## Procedure
- Verify all levels completed
- Update TASK states
- Archive completed items
- Update STATE.json

## Output Template
Closure Report:
- Completed Level
- Completed Items
- Archive Location
- Next Action

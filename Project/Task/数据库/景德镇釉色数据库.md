### 2. 白釉（甜白釉）

| **别名** | 甜白釉、葱根白 |
| **dynasty** | 明—清（永乐创烧） |
| **历史来源** | 甜白釉是永乐时期景德镇官窑创烧的白釉。永乐皇帝对白瓷情有独钟，据景德镇明代御窑厂遗址挖掘报告，永乐前期地层中 98% 以上的出土物为白瓷。白瓷历史悠久，从北朝白瓷萌芽、隋唐邢窑、宋代定窑至元代枢府卵白釉，明代永乐甜白釉达到白瓷巅峰 |

| **烧制温度** | 约 1250—1300℃ |

| **文化含义** | 永乐皇帝个人审美偏好的体现，代表白瓷制造技艺的最高峰，"万白之首"。与伊斯兰文化中崇尚白色的传统有关联，永乐时期郑和下西洋可能受到中东审美影响 |
| **代表作品** | 明永乐甜白釉暗花梅瓶（故宫博物院）；明永乐甜白釉僧帽壶；明永乐甜白釉暗花双龙纹碗 |
| **游戏化建议** | 类玉质感 PBR 材质：metallic 0、roughness 0.1~0.2、subsurface scattering 微启；温润乳白色（非纯白），轻微暖色调偏移 |

| **来源** | 百度百科 甜白釉/9792559；知乎《明永乐甜白釉何以成为白瓷巅峰》；中国科普网 kepu.gov.cn |

### 3. 青花

| 字段 | 数据 |
|------|------|
| **id** | GLAZE_003 |
| **glazeName** | 青花 |
| **别名** | 青花瓷、釉下蓝彩 |
| **dynasty** | 唐—清（元代成熟，明清鼎盛） |
| **历史来源** | 唐代已出现原始青花，元代景德镇创烧成熟青花瓷，明清两代达到鼎盛。青花瓷的制作技艺为国家级非物质文化遗产。钴料来源随时代变化：元用苏麻离青（进口，高铁低锰，呈色浓艳有铁锈斑），明成化用平等青（国产，发色淡雅），明嘉靖用回青（进口，蓝中泛紫），万历后用浙料（国产），清代沿用珠明料（云南产，发色鲜蓝） |
| **主要成分** | 着色剂为含氧化钴（CoO）的钴矿。制作工艺：瓷胎上用钴料描绘纹饰 → 罩透明釉 → 一次烧成。透明釉以石英、长石、粘土为主 |
| **烧制温度** | 约 1300—1330℃ |
| **烧制环境** | 还原焰，一次烧成（釉下彩），烧制约 10 小时，自然冷却 12—18 小时 |
| **颜色特征** | 钴料烧成后呈蓝色，着色力强、发色鲜艳、呈色稳定。不同钴料呈色差异：苏麻离青浓艳带铁锈斑；平等青淡雅柔和；回青蓝中泛紫；浙料蓝中泛灰；珠明料鲜蓝纯净 |
| **文化含义** | 中国瓷器的代表品种，"China"因瓷器得名而青花是其中最标志性的品类。兼具日用与陈设功能，是中国对外文化交流的重要载体，影响世界陶瓷史。元代青花大量出口至中西亚，影响伊斯兰陶瓷装饰风格 |
| **代表作品** | 元青花鬼谷子下山图罐（2005年伦敦拍卖2.3亿）；元青花萧何月下追韩信梅瓶（南京市博物馆）；明宣德青花海水白龙纹扁瓶（故宫博物院）；明成化青花缠枝莲纹碗 |
| **游戏化建议** | 青花为釉下彩，实现需双层渲染：底层白釉 + 上层蓝色纹饰（可程序化生成缠枝纹/云纹）；不同钴料对应不同蓝色调可作变体；纹饰可程序化参数控制复杂度 |
| **RGB**（钴蓝色） | (25, 50, 140) |
| **HSV**（钴蓝色） | H:227°, S:82%, V:55% |
| **来源** | 百度百科 景德镇传统青花瓷制作技艺/22842681；搜狗百科 v176025362；搜狐《景德镇四大名瓷之青花瓷》引13道工序 |

| **历史来源** | 起源于宋代的龙泉窑，是青釉派生釉色之一。宋代萌芽时期与东青釉属同类。明代永乐、宣德时期，龙泉窑试图还原宋龙泉的釉色，但釉色总体偏黄。明代景德镇窑也开始烧制豆青釉，至明代形成独立釉色体系。清代康熙、雍正、乾隆时期"仿龙泉"釉色接近纯"绿"，雍正时期最为出色，为豆青釉烧造巅峰 |

| **颜色特征** | 青中泛黄，色调如新鲜豆子，介于青与绿之间。明代以前呈色微近黄色，清代演变为趋近绿色且釉面凝厚。釉面光泽度弱于粉青和梅子青，温润淡雅 |

| **代表作品** | 宋龙泉窑青釉弦纹瓶 高31cm（故宫博物院）；明龙泉窑青釉凸花缠枝莲纹瓶 高33cm（故宫博物院）；明龙泉窑青釉刻花葡萄纹大盘 口径50.6cm（故宫博物院）；清雍正粉青釉七孔花插 高11.5cm（故宫博物院） |

## 五、烧制难度分级

| 难度 | 釉色 | 技术要点 | 烧制失败表现 |
|------|------|----------|-------------|
| 2 | 影青釉 | 铁含量控制需精准，铁过少则白而无青，铁过多则色深不雅 | 色深泛黄或无色发白 |
| 2 | 白釉 | 胎釉中铁杂质需极低；永乐甜白釉需特殊配釉实现乳浊感 | 发色偏黄或偏青 |
| 3 | 青花 | 钴料研磨细腻度控制；13道工序中画坯、混水、施釉需精准；不同钴料呈色差异需适配纹饰风格 | 钴料晕散、发色灰暗 |
| 3 | 豆青釉 | 铁含量需高于影青适中；还原气氛需严格控制以产生青黄微妙平衡 | 偏黄过甚或偏绿过冷 |
| 5 | 祭红釉 | 铜在高温下极易挥发，对窑温和气氛极度敏感（"十窑九不成"）；铜含量<1%的微量控制；还原焰必须精确无误 | 发色灰暗（氧化）、铜挥发无色（"烧飞"）、色调不均 |

## 六、游戏化实现建议汇总

| 釉色 | 材质策略 | 渲染参数 | 特殊机制 |
|------|----------|----------|---------|
| 影青釉 | 半透明类玻璃 | roughness 0.05, metallic 0.02, alpha 0.8 | 厚釉处颜色更深（厚度图驱动） |
| 白釉 | 类玉质感 | roughness 0.15, metallic 0, subsurface 微启 | 薄胎器物可透光（半脱胎效果） |
| 青花 | 双层渲染 | 底层白釉 + 上层蓝色纹饰层 | 纹饰可程序化生成（缠枝/云纹/海水）；钴料变体5种 |
| 祭红釉 | 深色釉面 | roughness 0.15, metallic 0.05 | 烧制难度最高，可设计为小概率成功事件（抽卡/培养机制） |
| 豆青釉 | 温润类玉质 | roughness 0.25, metallic 0 | 釉层厚度影响呈色——可关联拉坯器型的表面曲率 |

### 3. 梅瓶

| 字段 | 数据 |
|------|------|
| **id** | SHAPE_003 |
| **shapeName** | 梅瓶 |
| **dynasty** | 宋—清 |
| **用途** | 盛酒用具（宋元）；陈设瓷（明清） |
| **description** | 梅瓶因口径之小与梅之瘦骨相若而得名。瓶体修长，小口、短颈、丰肩、肩下渐收敛、圈足，有的附盖。宋代开始流行，元代除圆形外尚有八方形。明清器型从瘦长秀丽演变为肥矮丰硕，功能由日用渐变陈设。以景德镇窑制品最佳。 |
| **口部特征** | 小口、圆唇。口径极小，仅能插梅枝 |
| **颈部特征** | 短颈。颈肩之间形成近90度硬折角（典型特征） |
| **肩部特征** | 丰肩。造型线条丰满浑圆，向上高耸，为梅瓶最显著特征 |
| **腹部特征** | 肩下渐收敛，瓶体修长。宋代瘦长秀丽，明清肥矮丰硕。腹部为纹饰主体区域 |
| **底足特征** | 圈足，多为宽圈足。明永乐器宽圈足，明嘉万时期圈足外撇 |
| **difficulty** | 4 |
| **参考尺寸** | 明永乐：高28.5cm 口径4.8cm 足径11.3cm；明嘉靖：高44cm 口径6.5cm 足径13.8cm |
| **来源** | 故宫博物院 dpm.org.cn 词条239389、藏品226792；国家博物馆 chnmuseum.cn |

### 4. 玉壶春瓶

| 字段 | 数据 |

| **shapeName** | 玉壶春瓶 |
| **dynasty** | 隋—清 |
| **用途** | 酒器（宋元）；陈设瓷（明清） |
| **description** | 以"玉壶先春"得名，由唐代寺院净水瓶演变而来。基本形制为撇口、细颈、硕腹、圈足。以随时代变化而不同的弧线构成柔和匀称的瓶体，时代特征明显。宋代定窑、龙泉窑、景德镇窑、汝窑、耀州窑、钧窑均有烧制。明清渐变为陈设瓷，品种丰富。 |
| **口部特征** | 撇口，口沿外侈呈喇叭形展开 |
| **颈部特征** | 细长颈。为玉壶春瓶标志性特征，颈部长且细。明清颈部较宋代短粗 |
| **肩部特征** | 过渡平缓，颈腹间以柔和弧线衔接，无明显硬折肩 |
| **腹部特征** | 硕腹/圆腹下垂。腹部为全器最宽处，重心偏下，呈垂腹状。明清腹部较宋代更肥大 |
| **底足特征** | 圈足。足径适中，底足稳重。亦有矮足、平底者 |
| **difficulty** | 5 |
| **参考尺寸** | 明永乐：高27cm 口径7cm 足径10cm；元釉里红：高31.4cm 口径8.4cm；元釉里红松竹梅：高33.3cm 口径8.7cm 底径11.3cm；清乾隆白套红玻璃：高29.5cm 口径9.5cm 底径11.7cm |
| **来源** | 故宫博物院 dpm.org.cn 词条241111、百度百科引故宫藏品数据；首都博物馆藏景德镇窑釉里红花卉纹玉壶春瓶 |

| **description** | 罐的用途主要用以盛放或烹煮食物，造型特点是口径大腹丰且深，胫部内收，大底圈足。明清景德镇御窑罐品种丰富，有盖罐、将军罐等。将军罐直口短颈丰肩鼓腹，因宝珠顶纽盖形似将军头盔而得名，明代成化景德镇窑创制。 |

## 五、制作难度分级说明

| 难度 | 器型 | 拉坯技术要点 |
|------|------|-------------|
| 1 | 碗 | 简单旋转体，口腹一体，无颈肩过渡，初学者可成 |
| 2 | 盘 | 浅腹宽口，需控制底部平整度，口沿外翻需稳定手势 |
| 3 | 罐 | 大腹径需均匀鼓出，口肩过渡紧凑，需控制壁厚均匀 |
| 4 | 梅瓶 | 丰肩至瘦底需精确曲线控制，短颈硬折角为技术难点 |
| 5 | 玉壶春瓶 | 撇口→细颈→硕腹需完整S曲线控制，颈腹过渡弧度要求最高 |

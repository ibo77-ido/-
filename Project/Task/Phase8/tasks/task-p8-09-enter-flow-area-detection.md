# Task P8-09 Enter Flow Area Detection

## Goal

Map Phase6 workstations to Phase3 modules by area type.

## Scope

- Detect the current workstation area.
- Resolve the area to the matching module.
- Prevent invalid areas from opening a module.

## Acceptance

- Order, Shape, Glaze, and Firing each map to a unique entry path.
- Invalid area input does not start a session.

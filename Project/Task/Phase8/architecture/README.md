## Purpose

Phase8 splits the Phase6 × Phase3 runtime bridge design into small files so CodeBuddy can read and execute each piece with lower token cost.

## Execution Rule

Before any Phase8 implementation task, CodeBuddy must output a short design containing:
- Files Created
- Files Modified
- Scene Objects Changed
- Data Flow
- Acceptance Strategy

Implementation starts only after human approval.

# Task P9-02 Reference Map Strategy

## Goal

Decide how the reference-style map should be built in Unity.

## Scope

- Prefer the layered reference maps as the first visual foundation.
- Decide which layers are background-only.
- Decide which layers must be separate scene objects.

## Output

- A map strategy that says what gets stacked, what gets placed, and what stays decorative.

## Acceptance

- The map approach is fixed before scene work starts.

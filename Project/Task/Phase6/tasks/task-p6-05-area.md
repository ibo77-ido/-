# task-p6-05 Area

## 目标

实现区域配置、区域触发和区域管理。

## 步骤

1. 创建 `AreaConfigSO`。
2. 创建 `AreaTrigger`。
3. 创建 `AreaManager`。
4. 让区域触发进入和离开事件。
5. 让当前区域可查询。
6. 让区域映射到测试 UI。

## 验收

- 区域检测正常
- 进出区域不抖动
- 区域参数可配置
- 当前区域可查询

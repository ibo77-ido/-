## Purpose

Phase7 architecture files define the stage goal, boundaries, execution order, and exit criteria.

Implementation details belong in `Project/Task/Phase7/tasks/`.

## Execution Rule

Before executing any Phase7 implementation task, CodeBuddy must output a short design containing:
- Files Created
- Files Modified
- Scene Objects Changed
- Data Flow
- Acceptance Strategy

Implementation starts only after human approval.

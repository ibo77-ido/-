# Phase 11 音乐与音效接入建议

基于当前 Phase11 的 HUD、快捷寻路、相机操作与后续扩展计划，整理出建议接入的音乐与音效类型。

## 1. 当前判断

从项目现状来看，`Assets` 目录中暂未发现现成的 `.mp3`、`.wav`、`.ogg` 等音频资源。

因此这份文档先用于定义“需要哪些音频”，方便后续统一制作或采购。

## 2. 建议接入的音频类型

### 2.1 主游戏循环 BGM

用途：

- 常驻播放
- 覆盖主世界探索、HUD 浏览、日常经营过程

建议风格：

- 舒缓
- 带一点古风或手工经营感
- 不要太激烈，避免压过交互反馈

播放方式：

- 循环播放
- 全局单曲或少量循环曲目

优先级：

- P0

### 2.2 UI 交互音效

用途：

- 按钮点击
- 悬停反馈
- 面板打开 / 关闭
- 确认 / 取消操作

建议最小集合：

- 点击确认音
- 返回 / 取消音
- 禁用提示音
- 面板打开音
- 面板关闭音

播放方式：

- 非循环
- 短促、清晰

优先级：

- P0

### 2.3 寻路与自动交互音效

用途：

- 点击快捷入口后开始移动
- 到达目标建筑
- 触发自动交互
- 超时或失败取消

建议最小集合：

- 出发音
- 到达音
- 交互成功音
- 取消 / 失败音

播放方式：

- 非循环
- 偏轻量，不要太强烈

优先级：

- P0

### 2.4 相机操作反馈音

用途：

- 滚轮缩放
- 右键拖拽平移
- 自动复位开始
- 自动复位结束

建议最小集合：

- 缩放音
- 平移音
- 复位开始音
- 复位完成音

播放方式：

- 非循环
- 轻微 whoosh / click 类型即可

优先级：

- P1

### 2.5 场景区域差异化氛围音

用途：

- 接单区
- 拉坯区
- 施釉区
- 烧窑区
- 仓库区
- 材料区

建议说明：

- 这不是 Phase11 必需项
- 可用于后续增强区域辨识度
- 更适合在系统稳定后再加入

播放方式：

- 可作为区域环境音或区域 BGM 变化
- 建议后置

优先级：

- P2

## 3. 推荐接入顺序

### 第一阶段

- 主游戏 BGM
- UI 点击 / 确认 / 取消音

### 第二阶段

- 寻路开始 / 到达 / 成功 / 失败音

### 第三阶段

- 相机缩放 / 平移 / 复位反馈音

### 第四阶段

- 区域差异化氛围音

## 4. 建议的最小可用包

如果现在要做一个“最小可用”的音频接入包，建议先准备以下内容：

- 1 条主循环 BGM
- 3 到 5 个 UI 交互音效
- 4 个寻路与自动交互反馈音效

这样就足以覆盖 Phase11 的主要体验闭环。

## 5. 备注

1. 目前项目里未发现音频资源文件，建议后续统一补进 `Assets/Audio/` 或类似目录。
2. 如果项目后续需要更强的世界感，可以按区域再做 BGM 分层。
3. 音效与 BGM 尽量保持同一时代感和情绪基调，避免 UI 很现代、背景却很古典的割裂感。

## 1. 烧成流程总览

传统景德镇柴窑烧制经历 **5大阶段、8个子阶段**，总耗时约 **45~77小时**。

```
常温
  │  低温排水阶段 (8~12h)
  │  ├─ 预热排水: 常温→300°C
  │  └─ 晶体脱水: 300→600°C
  │
  │  素烧阶段 (8~12h)
  │  └─ 氧化烧清: 600→1000°C
  │
  ▲  玻化阶段 (8~12h)          ← 核心玩法
  │  ├─ 还原烧成: 1000→1200°C   ← 氧化→还原切换
  │  └─ 强还原保温: 1200→1280°C  ← 铁化学定型窗口
  │
  │  成熟阶段 (1~3h)
  │  └─ 高温成瓷: 1280→1320°C    ← 黄金窗口
  │
  ▼  冷却阶段 (34~46h)
     ├─ 急冷段: 1320→700°C
     └─ 缓冷段: 700→常温
```

### 关键温度节点

| 节点 | 温度 | 风险 | 评分影响 |
|------|------|------|----------|
| 水沸腾 | 100°C | 升温过快→炸裂 | 致命 |
| 低温段速率 | 常温→600°C | >100°C/h→裂纹 | 扣分 |
| 石英转化(α→β) | 573°C | 体积膨胀1%→应力裂纹 | 扣分 |
| 氧化→还原切换 | ~1000°C | 错过窗口→阴黄 | 核心扣分 |
| 釉料熔化 | 1200~1280°C | 温度不当→发色失/或釉泡 | 扣分 |
| 瓷化窗口 | 1280~1320°C | <1280→欠烧, >1320→过烧 | 核心评分 |
| 石英转化(β→α) | 573°C | 冷却过快→惊釉 | 扣分 |
| 方石英转化 | 226°C | 体积收缩2.5%→惊裂 | 致命 |

### 火焰颜色序列

按温度递增10个等级：

| 等级 | 火焰颜色 | 温度区间 | 气氛 | 关键事件 |
|------|----------|----------|------|----------|
| Lv1 | 暗红色 | 500~600°C | 氧化 | 首次可见火焰 |
| Lv2 | 红色 | 600~700°C | 氧化 | 预热完成 |
| Lv3 | 亮红色 | 700~800°C | 氧化 | 素烧开始, 573°C石英转化 |
| Lv4 | 樱红色 | 800~830°C | 氧化 | 有机物燃烧 |
| Lv5 | 亮樱红色 | 830~880°C | 强氧化 | 接近氛围切换 |
| Lv6 | 橘黄色 | 880~1050°C | 氧化→还原过渡 | **气氛切换关键点** |
| Lv7 | 暗黄色 | 1050~1150°C | 还原 | 还原气氛开始标志 |
| Lv8 | 亮黄色 | 1150~1250°C | 还原 | 釉料开始熔化 |
| Lv9 | 黄白色 | 1250~1300°C | 中性/弱还原 | 接近成熟 |
| Lv10 | 青白色/炽白 | 1300~1320°C | 中性 | 止火判断 |

### 气氛类型

| 气氛 | 特征 | 阶段 | 对瓷器的影响 |
|------|------|------|-------------|
| 强氧化 | 充分供氧, 炉火清明 | 素烧阶段 | 有机物烧清, 防止黑心 |
| 氧化 | 正常通风 | 低温阶段/冷却 | 基础燃烧 |
| 强还原 | 限氧, 烟气浓密, 火焰暗黄 | 玻化阶段 | Fe₂O₃→FeO呈青白, 青花发色 |
| 中性 | 供氧=消耗平衡 | 成熟阶段 | 瓷化完成, 釉面流平 |

## 2. FireScore 评分体系

### 总公式

```
FireScore = max(0, 100 - T - D - F - P)

T = 温度误差扣分 (0~35)
D = 烧成时长扣分 (0~20)
F = 火焰状态扣分 (0~20)
P = 缺陷扣分 (0~25, 致命缺陷强制归零)
```

满分 **100** 分。每次违规或失误从100分中扣减。最终 FireScore 不低于 0。

## 3. 温度误差评分 (T)

在 **6个关键检查点** 测量玩家实际温度与目标温度的偏差。

### 3.1 检查点定义

| 检查点 | 目标温度 | 允差 | 阶段 | 偏差每超出1°C扣分 | 最大扣分 |
|--------|----------|------|------|-------------------|----------|
| CP1 | ≤100°C/h 升温速率 | ≤100°C/h | 低温排水 | 超出部分 × 0.05 | 5 |
| CP2 | 573±20°C | ±20°C | 素烧(石英转化) | | 3 |
| CP3 | 1000±30°C | ±30°C | 氧化→还原切换 | | 7 |
| CP4 | 1280~1320°C | 窗口内 | 成熟阶段 | | 10 |
| CP5 | 573°C冷却 | 平滑过渡 | 急冷段 | | 3 |
| CP6 | 226°C | 不可早于226°C开窑 | 缓冷段 | | 7 |

### 3.2 评分细则

**CP1 — 低温升温速率 (0~5分扣除)**

低温段 (常温→600°C) 升温不得超过100°C/h。

| 条件 | 扣分 | 说明 |
|------|------|------|
| 全程 ≤ 100°C/h | 0 | 安全升温 |
| 部分>100°C/h 但≤130°C/h, 且持续<1h | 2 | 轻微超速，有风险但未出缺陷 |
| 全程出现过>130°C/h 或持续>1h | 5 | 严重超速，诱发裂纹/窑裂 |

> 数据来源：典型缺陷表"低温段升温过快→内部水沸腾产生蒸汽压力"、游戏化建议"升温速率范围低温段50-100°C/h, 偏离>15%警告, >30%触发缺陷"

**CP2 — 573°C石英转化点 (0~3分扣除)**

石英在573°C发生α→β晶型转变，体积膨胀约1%，必须平稳过渡。

| 条件 | 扣分 | 说明 |
|------|------|------|
| 573°C前后±20°C维持稳定（波动≤5°C） | 0 | 完美过渡 |
| 573°C前后波动5~10°C | 1 | 有微小风险 |
| 573°C前后波动>10°C 或 升温突变 | 3 | 石英转化不当，可能产生内应力 |

> 数据来源：阶段配置表"573℃石英转化点为关键节点"、典型缺陷表"石英转化体积突变"

**CP3 — 1000°C气氛切换点 (0~7分扣除)**

约1000°C是氧化焰→还原焰切换的关键窗口，直接影响铁的价态和瓷器呈色。这是**核心评分节点**。

| 条件 | 扣分 | 说明 |
|------|------|------|
| 在1000±20°C完成切换 | 0 | 完美时机 |
| 在1000±30°C完成切换 | 2 | 允许范围内 |
| 在1000±50°C完成切换 | 5 | 偏早或偏晚，影响发色 |
| 未切换（错过窗口）或切换温度>±50°C | 7 | 直接导致阴黄/发色不达标 |

> 数据来源：典型缺陷表"还原不足→铁以Fe₂O₃存在而非FeO"、游戏化建议"切换成功+20分, 失败发色不达标或阴黄"、阶段配置表"还原烧成1000~1200°C"

**CP4 — 1280~1320°C成熟窗口 (0~10分扣除)**

瓷器必须达到1280°C以上才能完全瓷化，但不能超过1320°C（过烧）。

| 条件 | 扣分 | 说明 |
|------|------|------|
| 最高温在1280~1320°C且保温≥1h | 0 | 完美成瓷 |
| >1320°C（过烧） | 10 | 过烧变形，严重报废 |
| <1280°C（欠烧） | 10 | 欠烧, 吸水率高, 声沉闷 |
| 1280~1320°C但保温<1h | 5 | 瓷化不充分 |
| 1280~1320°C但波动>10°C | 3 | 温差致局部缺陷 |

> 数据来源：阶段配置表"成熟阶段1280~1320°C, 1~3小时"、典型缺陷表"生烧<1280°C, 过烧>1320°C"、游戏化建议"黄金窗口1250-1280°C稳定保持"

**CP5 — 冷却阶段573°C (0~3分扣除)**

石英在冷却时573°C处发生β→α逆转化，体积收缩1%。冷却过快→惊釉。

| 条件 | 扣分 | 说明 |
|------|------|------|
| 573°C前后降温平缓(≤50°C/h) | 0 | 安全冷却 |
| 573°C前后降温50~100°C/h | 1 | 轻微风险 |
| 573°C前后降温>100°C/h | 3 | 诱发惊釉 |

**CP6 — 226°C方石英转化/开窑时机 (0~7分扣除)**

方石英转化产生2.5%体积收缩，是**冷却阶段最危险的节点**。

| 条件 | 扣分 | 说明 |
|------|------|------|
| 226°C完全通过前不接触冷空气 | 0 | 安全 |
| 226°C通过前短暂半开窑门 | 3 | 有惊裂风险 |
| 226°C通过前开窑或冷风进入 | 7 | 极大惊裂概率 |

> 数据来源：阶段配置表"226℃方石英转化最危险, 不可开窑门, 防止冷风进入造成惊裂"

**T 总分计算：**

```
T = CP1扣分 + CP2扣分 + CP3扣分 + CP4扣分 + CP5扣分 + CP6扣分
```

## 4. 烧成时间评分 (D)

对照 `阶段配置表` 中每个子阶段的**推荐持续时长**，检查实际烧成时间是否落在推荐范围内。

### 4.1 各阶段时长标准

| 阶段 | 子阶段 | 推荐时长 | 最短 | 最长 | 扣分上限 |
|------|--------|----------|------|------|----------|
| S1 | 预热排水 | 4~6h | 3h | 8h | 3 |
| S2 | 晶体脱水 | 4~6h | 3h | 8h | 3 |
| S3 | 氧化烧清 | 8~12h | 6h | 14h | 4 |
| S4 | 还原烧成 | 6~8h | 4h | 10h | 5 |
| S5 | 强还原保温 | 2~4h | 1h | 5h | 3 |
| S6 | 高温成瓷 | 1~3h | 0.5h | 3.5h | 4 |
| S7 | 急冷段 | 4~6h | 3h | 8h | 2 |
| S8 | 缓冷段 | 30~40h | 24h | 48h | 1 |

### 4.2 评分细则

每个阶段独立扣分，每超出推荐范围 **10%** 扣1分，超过50%取满扣分。

| 偏差百分比 | 扣分公式 | 说明 |
|-----------|----------|------|
| 0~10% | 0 | 在推荐范围内 |
| 10~20% | 1 | 轻微偏离 |
| 20~30% | 2 | 明显偏离 |
| 30~40% | 3 | 严重偏离 |
| 40~50% | 4 | 临界违规 |
| >50% | 阶段最大扣分 | 严重违规 |

**偏差百分比** = | 实际时长 - 推荐中值 | / 推荐中值 × 100%

> 数据来源：阶段配置表"持续时间"列、游戏化建议"偏离目标曲线>15%触发警告, >30%触发缺陷"（此处映射为扣分而非直接缺陷）

**D 总分计算：**

```
D = S1扣分 + S2扣分 + S3扣分 + S4扣分 + S5扣分 + S6扣分 + S7扣分 + S8扣分
```

## 5. 火焰状态评分 (F)

火焰颜色是传统柴窑"把桩师傅"判断温度的唯一依据。玩家需在10级火焰颜色过渡的**4个关键切换点**做出正确判断。

### 5.1 4个关键火焰切换点

| 切换点 | 从→到 | 对应温度 | 气氛变化 | 单次误判扣分 |
|--------|--------|----------|----------|-------------|
| FS1 | 亮红色→樱红色 | ~800°C | 氧化→强氧化 | 3 |
| FS2 | 橘黄色→暗黄色 | ~1050°C | 氧化→还原（核心） | 8 |
| FS3 | 暗黄色→亮黄色 | ~1150°C | 还原稳固 | 3 |
| FS4 | 黄白色→青白色/炽白 | ~1300°C | 弱还原→中性（止火判断） | 6 |

### 5.2 评分细则

**FS1 — 亮红→樱红 (0~3分扣除)**

素烧阶段从氧化进入强氧化，有机物开始大量燃烧。

| 条件 | 扣分 | 说明 |
|------|------|------|
| 正确识别樱红色出现并进行投柴调整 | 0 | 素烧节奏正确 |
| 延迟识别（温度已到830°C） | 1 | 轻微滞后 |
| 判定错误（以为还在亮红阶段） | 3 | 升温节奏失控 |

**FS2 — 橘黄→暗黄 (0~8分扣除) — 核心评分项**

这是氧化→还原的**最关键切换点**。暗黄色火焰+窑内烟气增多=还原气氛开始标志。

| 条件 | 扣分 | 说明 |
|------|------|------|
| 在橘黄→暗黄渐变时立即切换还原气氛 | 0 | 完美时机 |
| 暗黄色明显后1级延迟内切换 | 3 | 轻微延迟，少量烟气损失 |
| 暗黄色明显后2级以上延迟或提前切换 | 6 | 严重时机失当 |
| 未识别暗黄色（始终未切换还原） | 8 | 完全失败, 瓷器发黄 |

> 数据来源：火焰颜色对照表"暗黄色还原气氛开始标志, 把桩师傅凭此判断还原是否到位"、游戏化建议G03"切换成功+20分, 失败发色不达标"

**FS3 — 暗黄→亮黄 (0~3分扣除)**

还原气氛稳定后火焰转为明亮黄色，釉料开始熔化。

| 条件 | 扣分 | 说明 |
|------|------|------|
| 正确识别，维持还原气氛 | 0 | 还原烧成顺利进行 |
| 判别迟疑或判断错误 | 3 | 可能还原强度波动 |

**FS4 — 黄白→青白/炽白 (0~6分扣除) — 止火判断**

火焰转为青白/炽白表示已达1320°C附近，需要判断"止火"时机。这是把桩师傅最难的技术。

| 条件 | 扣分 | 说明 |
|------|------|------|
| 青白/炽白出现后≥1270°C立即止火 | 0 | 完美掌控成熟度 |
| 青白/炽白出现后延迟止火（过烧风险） | 4 | 可能局部过烧 |
| 未识别青白/炽白, 提前止火（>1320°C未到） | 6 | 可能欠烧或温度不足 |
| 完全忽视火焰颜色, 随意止火 | 6 | 严重失误 |

**F 总分计算：**

```
F = FS1扣分 + FS2扣分 + FS3扣分 + FS4扣分
```

## 6. 缺陷扣分规则 (P)

烧制过程中因失误触发的实际缺陷将直接扣分。数据来源为 `典型缺陷表`。

### 6.1 缺陷分级

| 等级 | 定义 | 触发条件示例 |
|------|------|-------------|
| **致命** (Fatal) | 瓷器完全报废, 分数强制归零 | 窑裂/碎坯、严重过烧塌陷粘连 |
| **严重** (Severe) | 大幅降低器物品级, -20分 | 欠烧、严重烟熏+阴黄复合 |
| **中等** (Moderate) | 品相受损, -8~-12分 | 变形、起泡、阴黄、窑粘、失透 |
| **轻微** (Minor) | 表面瑕疵, -5分 | 针孔 |

### 6.2 缺陷扣分表

| 编号 | 缺陷名称 | 表现 | 发生阶段 | 扣分 | 是否可叠加 |
|------|----------|------|----------|------|-----------|
| D1 | 开裂/窑裂 | 坯体裂纹、破损、贯通裂 | 低温排水/冷却 | **归零** | 致命，不可叠加 |
| D2 | 过烧(严重) | 坯体塌陷/粘连/流淌 | 成熟(>1320°C严重超标) | **归零** | 致命，不可叠加 |
| D3 | 生烧/欠烧 | 吸水率高、声沉闷、釉面无光 | 成熟(<1280°C) | 20 | 是 |
| D4 | 过烧(轻度) | 轻微变形、釉面轻微流淌 | 成熟(略超1320°C) | 15 | 是 |
| D5 | 变形 | 器型歪扭不圆 | 玻化/成熟 | 10 | 是 |
| D6 | 阴黄/发黄 | 瓷器表面黄色调, 白度不足 | 还原阶段 | 10 | 是 |
| D7 | 起泡/釉泡 | 釉面气泡凸起 | 玻化/成熟 | 8 | 是 |
| D8 | 窑粘 | 器物与匣钵/垫具粘连 | 成熟 | 8 | 是 |
| D9 | 烟熏 | 釉面灰黑, 白度丧失 | 还原阶段 | 7 | 是 |
| D10 | 无光/失透 | 釉面哑光无光泽 | 冷却阶段 | 10 | 是 |
| D11 | 针孔 | 釉面细小针尖状凹陷 | 玻化/成熟 | 5 | 是 |

> 数据来源：典型缺陷表全部10行 + 游戏化建议"严重惩罚: 碎坯→完全报废, 得分归零; 欠烧价值降低80%; 变形价值降低50%"

### 6.3 缺陷触发逻辑

缺陷不是随机触发的。每个缺陷都有明确的**触发条件映射**：

| 缺陷 | 由哪些检查点违规触发 |
|------|--------------------|
| D1 开裂/窑裂 | CP1低温升温过速(>130°C/h) **或** CP6提前开窑 |
| D2 过烧(严重) | CP4 T>1320°C 持续>0.5h |
| D3 欠烧 | CP4 T<1280°C |
| D4 过烧(轻度) | CP4 1320°C<T≤1330°C 或 FS4延迟止火 |
| D5 变形 | CP4 T>1320°C 短暂触及 或 窑内温差>±10°C |
| D6 阴黄 | CP3还原切换失败/过晚 或 FS2未识别暗黄 |
| D7 起泡 | CP4升温过快(>50°C/h在1200~1300区间) 或 S4时长不足 |
| D8 窑粘 | CP4 T≥1325°C 或 S6时长过长(>4h) |
| D9 烟熏 | FS2还原过浓(烟气滞留过长) 或 S4时长过长 |
| D10 失透 | CP5/CP6冷却过慢在700-900°C区间 |
| D11 针孔 | CP4温度偏低(1280-1300°C) 且保温不足 或 S6时长不足 |

> 实现时使用 AND/OR 布尔逻辑组合多个检查点违规结果

### 6.4 复合缺陷规则

多个缺陷可以同时触发，扣分累加，但上限为25分（致命缺陷除外）。

```
P = sum(触发的缺陷扣分)
P = min(P, 25)  // 除非致命缺陷
P = 归零 if D1或D2触发  // 致命缺陷覆盖一切
```

### 6.5 缺陷对T/D/F的联动影响

缺陷发生时，除了P直接扣分外，还可能改变T/D/F的计算：

| 缺陷 | 联动影响 |
|------|---------|
| D1 开裂 | FireScore = 0，不再计算其他分数 |
| D2 过烧(严重) | FireScore = 0 |
| D3 欠烧 | CP4强制取满扣分(10) |
| D6 阴黄 | FS2强制取满扣分(8) |
| D10 失透 | CP5强制取满扣分(3) |

> 设计原则：缺陷是"症状"——它发生意味着对应的检查点已经违规。所以缺陷扣分(P)与温度/火焰扣分(T/F)都是对同一失误的惩罚，但两个维度不重复（P计入缺陷维，T/F计入误差维）。致命缺陷直接短路。

## 7. 完整 FireScore 计算公式

```
// Step 1: 计算基础扣分
T = CP1 + CP2 + CP3 + CP4 + CP5 + CP6        // 0~35
D = S1 + S2 + S3 + S4 + S5 + S6 + S7 + S8    // 0~20
F = FS1 + FS2 + FS3 + FS4                     // 0~20

// Step 2: 缺陷判定
if (D1 or D2 triggered):
    FireScore = 0
    STOP

P = sum(triggered defect penalties)           // 0~25, capped
P = min(P, 25)

// Step 3: 最终得分
FireScore = max(0, 100 - T - D - F - P)
```

### 评分等级

| 分数区间 | 等级 | 名称 | 描述 |
|----------|------|------|------|
| 96~100 | S | 天工 | 完美烧成，器物品级绝佳 |
| 85~95 | A | 上品 | 工艺精湛，仅有微小瑕疵 |
| 70~84 | B | 中品 | 合格成品，有一定瑕疵 |
| 50~69 | C | 次品 | 品相受损，但仍可使用 |
| 1~49 | D | 废品 | 严重缺陷，仅供经验累积 |
| 0 | F | 碎坯 | 完全报废 |

## 10. 边界案例

### 10.1 上限边界 — 完美烧成

```
条件：所有操作完美，无任何违规
FireScore = 100
等级：S级"天工"（注意：实际游戏中极难达到）
```

### 10.2 下限边界 — 窑裂归零

```
条件：D1(窑裂)或D2(严重过烧)触发
FireScore = 0
等级：F级"碎坯"
```

### 10.3 临界边界 — 欠烧阈值

```
条件：CP4温度=1279°C（差1°C达标）
判定：欠烧触发(因<1280°C)

vs CP4温度=1280°C（恰好达标）
判定：不触发欠烧

→ 1280°C是精确分界线
```

### 10.4 临界边界 — 过烧阈值

```
条件：CP4温度=1320°C（上限内）
判定：完美成瓷

vs CP4温度=1321°C（超出1°C）
判定：扣分(CP4=10)

→ 1320°C是精确分界线
```

### 10.5 临界边界 — 缺陷扣分上限

```
条件：同时触发D3(20)+D5(10)+D6(10)+D7(8)+D8(8)
原始扣分：20+10+10+8+8=56
实际扣分：cap=25

→ P上限为25，即使多个缺陷叠加
```

### 10.6 边界 — 最低非零分（纯缺陷触发，无检查点违规）

```
假设：所有检查点都OK(T=0, D=0, F=0)，但因特殊原因触发了针孔(D11, P=5)
FireScore = 100 - 0 - 0 - 0 - 5 = 95 → A级

→ 缺陷可能在检查点"看起来OK"时仍然触发（如釉料配方本身的问题）
```

### 10.7 边界 — 最大理论扣分（非致命）

```
T=35(全部检查点最大) + D=20(全部阶段最长偏离) + F=20(全部火焰误判) + P=25(缺陷叠加上限)
= 100

FireScore = max(0, 100-100) = 0 → F级

但这个极端情况若包含致命缺陷D1/D2，直接归零。
```

## 附录 C：V1.1 优化 — 单一错误单次处罚原则

### C.1 问题识别

当前 FireScore 存在 **双重处罚** 风险：

| 场景 | 违规 | CP扣分(T) | 缺陷扣分(P) | 总扣分 | 问题 |
|------|------|-----------|-------------|--------|------|
| 1000°C还原失败 | CP3: -7 | D6阴黄: -10 | **-17** | 同一失误被计算两次 |
| 过烧 | CP4: -10 | D2过烧: fatal | 归零 | 致命缺陷覆盖，OK |
| 升温过快 | CP1: -5 | D1窑裂: fatal | 归零 | 致命缺陷覆盖，OK |

**CP3 + D6 是被双重处罚的典型案例**：
- CP3 = "未在1000°C切换还原" (-7)
- D6 阴黄 = "还原不足导致铁未还原" (-10)
- 两者是 **同一因果链**：还原失败 → 阴黄

### C.2 PenaltySource 机制

引入 `PenaltySource` 概念，确保每个根本原因只被处罚一次：

```
每个玩家失误映射到一个 PenaltySource
├── PenaltySource → CP检查点 (温度/时长/火焰)
└── PenaltySource → DefectTrigger (缺陷触发)

规则：CP检查点处罚 与 同源的缺陷处罚 不叠加
     取 max(CP扣分, 缺陷扣分)
```

### C.3 修正后的处罚映射

| PenaltySource | CP扣分 | 可能触发的缺陷 | V1.1扣分(取max) |
|---------------|--------|---------------|-----------------|
| 低温升温过快 | CP1: 0~5 | D1窑裂(fatal) | 若D1触发→fatal；否则5 |
| 573°C石英转化不当 | CP2: 1~3 | — | CP2(3) |
| 还原切换失败/过晚 | CP3: 2~7 | D6阴黄(-10) | **max(7, 10)=10** |
| 成熟温度不足 | CP4: 0~10 | D3欠烧(-20) | **max(10, 20)=20** |
| 成熟温度过高 | CP4: 10 | D2/D4过烧 | 若D2→fatal；若D4→max(10,15)=15 |
| 冷却过快(573°C) | CP5: 1~3 | D10失透(-10) | **max(3, 10)=10** |
| 提前开窑(226°C) | CP6: 3~7 | D1窑裂(fatal) | 若D1→fatal；否则CP6(7) |
| 保温时间不足 | S1~S8: 各阶段 | D3欠烧/D11针孔 | **max(D扣分, 缺陷扣分)** |
| 火焰误判(FS2) | FS2: 3~8 | D6阴黄(-10) | **max(8, 10)=10** |
| 冷却过慢(700~900°C) | S7/S8: 1~3 | D10失透(-10) | **max(3, 10)=10** |
| 还原过浓/烟气滞留 | — | D9烟熏(-7) | 7(无CP冲突) |

### C.4 修正后的 FireScore 公式（V1.1）

```
FireScore = max(0, 100 - T_unique - D_unique - F_unique - P_unique)

其中 T_unique/D_unique/F_unique 已排除被P覆盖的PenaltySource
P_unique 取与原CP扣分的 max 值
致命缺陷(D1/D2严重) → 直接归零
```

### C.5 修正前后对比

| 案例 | V1.0 扣分 | V1.1 扣分 | 差 |
|------|-----------|-----------|-----|
| 还原失败→阴黄 | CP3(7)+D6(10)=17 | max(7,10)+unique=10 | -7 |
| 温度不足→欠烧 | CP4(10)+D3(20)=30 | max(10,20)+unique=20 | -10 |
| 冷却过快→失透 | CP5(3)+D10(10)=13 | max(3,10)+unique=10 | -3 |
| 升温快→窑裂 | CP1(5)+D1(fatal)=fatal | D1(fatal)=fatal | 无变化 |

> V1.1 更加公平，玩家不会因为同一因果链被双重扣分。挫败感显著降低。

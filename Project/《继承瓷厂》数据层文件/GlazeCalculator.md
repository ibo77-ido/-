## 2. 距离算法

### 欧几里得距离 (Euclidean Distance)

```
GlazeScore(i) = sqrt( (Cu_player - Cu_i)^2 + (Fe_player - Fe_i)^2 + (Co_player - Co_i)^2 )
```

| 参数 | 含义 |
|------|------|
| i | 釉色索引 (Yingqing / SweetWhite / BlueWhite / Jihong / Dongqing) |
| Cu_player | 玩家输入的铜含量 |
| Fe_player | 玩家输入的铁含量 |
| Co_player | 玩家输入的钴含量 |
| Cu_i, Fe_i, Co_i | 釉色 i 的模板向量 |

### 选择理由

- 3 种元素构成三维向量，欧几里得距离是最自然的几何距离
- 各元素取值范围相近 (0~0.02)，无需加权即可公平比较
- 物理意义明确：距离越小，化学配方越接近

### 公式简化

由于只需比较大小，可省略开方运算：

```
GlazeScore_sq(i) = (Cu_player - Cu_i)^2 + (Fe_player - Fe_i)^2 + (Co_player - Co_i)^2
```

取 min(GlazeScore_sq) 即为最佳匹配釉色。

## 3. Excel 实现公式

假设 Excel 布局：

| 单元格 | 内容 |
|--------|------|
| B2 | 玩家 Cu |
| B3 | 玩家 Fe |
| B4 | 玩家 Co |
| D2:D6 | 模板 Cu (5行) |
| E2:E6 | 模板 Fe (5行) |
| F2:F6 | 模板 Co (5行) |

**平方误差公式（推荐，避免 SQRT 精度问题）:**

```excel
=(B$2-D2)^2 + (B$3-E2)^2 + (B$4-F2)^2
```

**完整欧几里得距离公式:**

```excel
=SQRT((B$2-D2)^2 + (B$3-E2)^2 + (B$4-F2)^2)
```

**最佳匹配（取最小距离的釉色名）:**

```excel
=INDEX(釉色名范围, MATCH(MIN(距离范围), 距离范围, 0))
```

## 5. 错误案例与局限性

### 5.1 已知失败案例

| 编号 | 描述 | Cu | Fe | Co | 算法输出 | 正确结果 | 原因 |
|------|------|----|----|----|----------|----------|------|
| FC-01 | 完美影青输入 → 可能输出冬青 | 0.00 | 0.020 | 0.00 | Yingqing 或 Winter Green | 两者化学相同 | Cu/Fe/Co 无法区分 |
| FC-02 | 完美冬青输入 → 可能输出影青 | 0.00 | 0.020 | 0.00 | Winter Green 或 Yingqing | 两者化学相同 | Cu/Fe/Co 无法区分 |

### 5.2 数学极限边界

| 场景 | 输入 | 最近釉色 | 距离 | 次近釉色 | 距离差 |
|------|------|----------|------|----------|--------|
| 甜白↔影青中点 | Fe=0.0175 | Sweet White | 0.002500 | Yingqing | 0.000000 (tie) |
| 影青↔霁红最近点 | Cu=0.005, Fe=0.010 | Yingqing | 0.011180 | Sacrificial Red | 0.011180 |
| 青花↔霁红边界 | Cu=0.005, Co=0.010 | Blue-and-White | 0.011358 | Sacrificial Red | 0.000000 (tie) |

### 5.3 根本原因

影青釉与冬青釉的配方差异在于**烧制工艺**（温度曲线、还原气氛）而非原料比例。仅凭 Cu/Fe/Co 三元素无法区分。这是算法模型的理论上限，非实现缺陷。

**建议补救方案（不修改数据库结构的前提下）：**

- 算法层面：当 Cu/Fe/Co 向量与影青/冬青距离同时为最近时（即距离相等），结合 TemperatureMin/TemperatureMax 模板差异做二次判定
  - 冬青釉: 1250~1300°C
  - 影青釉: 1250~1280°C
- 游戏层面：系统可随机输出两者之一，或提示"您调制出了介于影青与冬青之间的釉色"

## 7. 结论

| 维度 | 评估 |
|------|------|
| Cu/Fe/Co 三元素区分能力 | 5种釉色中可完美区分 4 种：青花(Co)、霁红(Cu)、甜白(Fe=0.015)、影青/冬青组(Fe=0.020) |
| 铁系组内区分 | 甜白(Fe=0.015) 可与影青/冬青(Fe=0.020) 明确区分，安全边界 Fe=0.0175 |
| 影青↔冬青 | 化学向量完全相同，Cu/Fe/Co 模型无法区分 — 属于数据层面的固有限制 |
| 青花↔霁红边界 | Cu=0.005, Co=0.010 是理论平局点，两侧均可被正确识别 |
| 推荐方案 | 主算法使用欧几里得距离；当影青/冬青平局时，以烧制温度做二级裁决 |
